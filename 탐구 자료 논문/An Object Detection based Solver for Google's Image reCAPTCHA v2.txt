해당 연구에서는 Google의 이미지 기반 reCAPTCHA v2를 우회하기 위해 고급 객체 탐지 기술을 사용한 완전 자동화된 시스템을 제안했다. 주요 포인트는 다음과 같다.

1. 해당 연구에서는 reCAPTCHA v2의 이미지 챌린지를 해결하기 위해 깊은 신경망(Deep Neural Network, DNN) 이미지 분류기와 상용 이미지 인식 서비스에서 제공하는 비전 API를 사용하는 자동 프로그램이 가능함을 보여 주었다. 이를 방지하기 위해 Google은 등장하는 위협에 대응하여 reCAPTCHA v2의 이미지 챌린지에 상당한 업데이트를 하여 이전 접근법들이 크게 비효율적이 되도록 만들었다.

2. 이 논문은 최신 버전의 reCAPTCHA v2에 대한 고급 객체 탐지 기반 해결 시스템의 강도를 조사한다. 논문에서 기재된 시스템은 83.25%의 온라인 성공률로 reCAPTCHA v2의 가장 발전된 챌린지를 해결할 수 있는 완전 자동화된 객체 탐지 기반 시스템이다.

3. 또한, 반인식 메커니즘, 향상된 반봇 탐지 기술, 조정 가능한 보안 선호도와 같은 reCAPTCHA v2의 갱신된 보안 기능들을 연구한다. 광범위한 실험을 통해 이 보안 기능들이 자동화된 공격에 대한 어느 정도의 저항력을 제공할 수 있지만, 적들이 대부분을 우회할 수 있음을 보여준다.

4. 결론적으로 실험 결과는 객체 탐지 기술의 최근 진보가 단순 객체 탐지를 기반으로 하는 이미지 CAPTCHA 설계의 보안에 심각한 위협이 될 수 있음을 나타낸다. 따라서 이 시스템은 깊은 신경망을 사용하여 이미지 내의 객체를 식별하고 분류하는 고도로 훈련된 모델을 기반으로 하며, 이는 reCAPTCHA v2가 제시하는 이미지 챌린지를 효과적으로 해결할 수 있음을 시사하고 있다.​
